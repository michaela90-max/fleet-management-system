# Fleet Management System

## Overview

This project is an object-oriented Java application that manages a fleet of sailing and power boats. The program allows users to add and remove boats, track maintenance expenditures, and manage fleet data while ensuring maintenance costs do not exceed each boat's purchase price.

## Features

- Add new boats to the fleet
- Remove boats from the fleet
- View the current fleet inventory
- Track maintenance expenses
- Prevent maintenance spending from exceeding purchase price
- Save and load fleet data using file serialization

## Technologies

- Java
- Object-Oriented Programming (OOP)
- ArrayLists
- File I/O
- Serialization

## Project Structure

- `FleetApp.java` – Main application and user interface
- `FleetManager.java` – Fleet management logic
- `Boat.java` – Boat object and associated methods

- Object-oriented software design
- Data management
